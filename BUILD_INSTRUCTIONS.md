# كيفية استخراج ملف APK وتحميله على هاتفك

بما أنني ذكاء اصطناعي، لا يمكنني إرسال ملف `.apk` مباشرة. ولكن يمكنك بناؤه بسهولة باستخدام الكود الذي وفرته لك في مجلد `flutter_app_source`.

## الخطوة 1: تجهيز التطبيق (الويب)
1. قم برفع كود React (الملفات الموجودة في هذا المشروع) على استضافة مجانية وآمنة (HTTPS) مثل **Vercel** أو **Netlify**.
   * هذا ضروري لأن ميزة "تحويل الصوت لنص" تتطلب اتصالاً آمناً لتعمل على الجوال.
2. انسخ رابط موقعك الجديد (مثلاً: `https://my-speech-app.vercel.app`).

## الخطوة 2: إعداد مشروع Flutter
1. حمل وثبت [Flutter SDK](https://docs.flutter.dev/get-started/install) على جهاز الكمبيوتر الخاص بك.
2. أنشئ مشروع جديد في التيرمينال (Terminal/CMD):
   ```bash
   flutter create speech_app
   ```
3. افتح المجلد الجديد `speech_app`.

## الخطوة 3: نسخ الملفات
قم بنسخ محتويات الملفات التي وفرتها لك في هذا الشات إلى أماكنها الصحيحة داخل مجلد `speech_app`:
1. انسخ محتوى `flutter_app_source/pubspec.yaml` واستبدل محتوى ملف `pubspec.yaml` الموجود لديك.
2. انسخ محتوى `flutter_app_source/lib/main.dart` واستبدل محتوى ملف `lib/main.dart` الموجود لديك.
   * **مهم:** ابحث عن السطر `https://YOUR-APP-URL.vercel.app` في الملف وضع الرابط الخاص بك من الخطوة 1.
3. انسخ محتوى `flutter_app_source/android/app/src/main/AndroidManifest.xml` واستبدل الملف الموجود في `android/app/src/main/AndroidManifest.xml`.

## الخطوة 4: بناء ملف APK (استخراجه)
1. افتح التيرمينال داخل مجلد المشروع.
2. نفذ الأمر التالي لتحميل المكتبات:
   ```bash
   flutter pub get
   ```
3. نفذ الأمر التالي لبناء ملف التطبيق:
   ```bash
   flutter build apk --release
   ```

## الخطوة 5: نقل الملف للهاتف
1. بعد انتهاء البناء، ستجد الملف في المسار التالي داخل مجلد المشروع:
   `build/app/outputs/flutter-apk/app-release.apk`
2. قم بنقل هذا الملف إلى هاتفك عبر كابل USB أو رفعه على Google Drive وتحميله.
3. قم بتثبيته على هاتفك واستمتع بالتطبيق!
